HN-Lang: Hindi-Inspired Programming

Welcome to HN-Lang – a programming language designed especially for Hindi speakers! HN-Lang makes coding more intuitive by using Hindi words and their transliterations as keywords, instead of English-based Python syntax. It’s beginner-friendly and runs fully on Python via a translation layer.
